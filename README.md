# CHESS PERFORMANCE COMPARATOR

This project compares a Chess.com user's performance when playing the **Ruy Lopez**, **Queen's Gambit** or **English Opening** as White against the results of three of the greatest players in history: **Mikhail Tal**, **Anatoly Karpov** and **Garry Kasparov**.

The project consists of 4 scripts (written with Catalan comments and variable names) organized as follows:

1. **Historical Database:** First of all, we obtain the results of the chosen GM's with the chosen openings from [chessgames.com](https://www.chessgames.com/) and [introduce them into MySQL](01_dades_gms.sql)

2. **Importing User's Games:** Next, we connect to [chess.com](https://www.chess.com) and [download](02_importar_chesscom.py) all the user's games into a PGN file.

3. **Processing the PGN and Computing The User's Perfomance:** Then, we filter all the games from the PGN file to select only the ones where the users plays as White any of the three openings mentioned, [compute the performance](03_rendiment.py) and connect to MySQL to update the database to store user's data. 

4. **Comparing the Performance:** Finally, we extract the results from the database to [calculate the affinity](04_comparador.py) between our results and those of the GM's. Additionally, we generate a chart to visually represent the results.

The file [Resultats_finals.png](Resultats_finals.png) shows the chart generated by running the project's scripts.

**For privacy reasons, the sample username remains anonymous. To run the project, you must replace the comments **#AFEGIR USUARI** from scripts [02](02_importar_chesscom.py) and [03](03_rendiment.py) by the desired username to be studied.** **You will have to update the MySQL database connection details with your own credentials.**

# 👑 COMPARADOR DE RENDIMENTS EN ESCACS

En aquest projecte, comparem els resultats d'un usuari de Chess.com quan juga les obertures Ruy López, Gambit de Dama o Anglesa, amb els resultats de 3 dels millors jugadors de la història, **Mikhail Tal**, **Anatoly Karpov** i **Garry Kasparov** utilitzant aquestes mateixes obertures. 

El projecte consta de 4 scripts organitzats de la següent manera:

1. **Base de dades històriques:** El primer pas ha estat obtenir els resultats dels jugadors històrics a partir de la web [chessgames.com](https://www.chessgames.com/) i [introduir a MySQL aquestes dades](01_dades_gms.sql).

2. **Importar les partides de l'usuari:** A continuació, accedim a [chess.com](https://www.chess.com) i [descarreguem](02_importar_chesscom.py) totes les partides de l'usuari en un fitxer pgn.

3. **Processar el PGN i calcular el rendiment de l'usuari:** Després filtrem d'entre totes les partides del fitxer pgn aquelles on hem jugat amb blanques alguna de les 3 obertures, [determinem el rendiment propi amb cada](03_rendiment.py) i ens connectem amb la nostra base de dades per introduir les nostres dades. 

4. **Comparar rendiments:** Per últim, extreiem les dades de la base de dades, [calculem la similitud](04_comparador.py) dels nostres resultats amb els dels 3 jugadors escollits i generem un gràfic per representar de manera visual els resultats.

El fitxer [Resultats_finals.png](Resultats_finals.png) mostra els resultats obtinguts després d'executar els diferents scripts.

**El nom d'usuari es mantindrà anònim en aquesta publicació. Als scripts [02](02_importar_chesscom.py) i [03](03_rendiment.py) caldrà substituir el comentari #AFEGIR USUARI per un usuari real de la plataforma per tal que s'executin correctament.** **Addicionalment, caldrà modificar les dades de la connexió a la base de dades per les pròpies de cada usuari.**
